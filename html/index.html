H

Helix Preset Viewer

A free, open-source visualizer for Line 6 Helix / HX Stomp / HX Effects preset files. Upload a .hlx file to explore the signal chain, block parameters, snapshots, and footswitch assignments. Runs entirely in your browser.

Built with Claude Code (AI-assisted). May contain bugs or inaccuracies - please report issues on GitHub.
